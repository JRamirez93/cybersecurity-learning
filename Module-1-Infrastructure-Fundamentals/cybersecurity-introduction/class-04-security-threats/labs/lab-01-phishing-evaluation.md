# Lab 01 – Phishing Evaluation and Malicious Site Reporting

## Description

In this lab, I learned how to **identify potential phishing attempts** and report malicious websites.  
The objective was to recognize indicators of phishing and understand the process of contributing to security communities.

---

## Lab Walkthrough

### Step 1 – Access Phishing Quiz

1. Navigate to the Phishing Quiz link provided by the instructor.  
2. Carefully read each message presented and evaluate whether it could be a phishing attempt.

### Step 2 – Analyze Indicators

Look for typical phishing signs, including:

- Legitimacy of the sender  
- Requests for sensitive information  
- Spelling or grammar errors  
- Suspicious URLs or links  

Document observations in a notes file.

![Phishing Quiz Step](screenshots/lab1/Step1.png)

### Step 3 – Report Malicious Sites (Optional)

1. If a phishing site is identified, follow the reporting procedure to contribute to the security community.  
2. Submit the site through the reporting form in the quiz or associated platform.
