# Class 04 – Security Threats

Notes for **Class 04 – Security Threats** in Cybersecurity Introduction.  
Beginner-friendly explanations included for each type of threat.

---

## 1. Threats and Vulnerabilities

- **Malware** – Malicious software including viruses, worms, and trojans.  
- **Ransomware** – Malware that encrypts files and demands payment.  
- **Bombs and Logic Bombs** – Code that triggers harmful actions under certain conditions.  
- **Rootkits** – Tools that hide malware or attackers on systems.  
- **Zero-Day Attacks** – Exploits unknown vulnerabilities.

---

## 2. Social Engineering

- Manipulating people to bypass security controls.  
- Examples: phishing emails, spoofing, pretexting.  

---

## 3. Network Attacks

- **DoS (Denial of Service)** – Flooding a service to make it unavailable.  
- **DDoS (Distributed Denial of Service)** – Coordinated attacks from multiple systems.  
- **Backdoors** – Hidden access points for attackers.  
- **Replay Attacks** – Reusing captured data to bypass security.

---

## 4. Web and Code Exploits

- **Web Attacks** – Target web applications.  
- **Remote Code Execution (RCE)** – Attacker runs code on a victim system.  
- **SQL Injection** – Exploiting databases via malicious queries.  
- **Cross-Site Scripting (XSS)** – Injecting malicious scripts into web pages.
