# Lab 02 – Bitwarden Password Manager

## Description

In this lab I used Bitwarden to create and manage a cloud-based password vault.
The objective was to understand how password managers store credentials securely and synchronize them across devices.

---

## Actions Performed

1. Created a Bitwarden account.
2. Generated a password vault.
3. Stored several test credential entries.
4. Installed the Bitwarden mobile application.
5. Verified that the stored credentials were accessible from the mobile device.
6. Removed the test entry after verification.

---

## Example Entries

| Service         | Username                                    |
| --------------- | ------------------------------------------- |
| Email Test      | [user@example.com](mailto:user@example.com) |
| Web Account     | test_user                                   |
| Internal System | admin_test                                  |

---

## Key Takeaways

* Cloud password managers allow secure credential storage across multiple devices.
* Password vault synchronization enables access from desktop and mobile environments.
* Using a password manager helps reduce password reuse and improves credential management.
