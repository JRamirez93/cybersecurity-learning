# Cybersecurity Learning Roadmap

This repository documents my learning journey in infrastructure, networking, and cybersecurity.

The content is organized into progressive modules that include:

- theoretical concepts learned during courses
- hands-on labs
- technical challenges
- integrative projects

The goal of this repository is not only to reinforce my professional profile by documenting what I learn, but also to provide a clear roadmap for people who want to start learning cybersecurity or better understand this field.

This project is currently **under development**, and the modules will be progressively completed.

---

# Repository Goals

- Document my technical progress in infrastructure and cybersecurity.
- Consolidate knowledge through labs and practical projects.
- Provide a structured learning roadmap for people interested in cybersecurity.
- Share technical notes and learning resources.

# Module Overview

## Module 1 — Infrastructure Fundamentals (NOC Foundation)

Objective: understand the fundamentals of operating systems, networking, and security.

Contents:

- Introduction to Linux
- Introduction to Networking
- Introduction to Cybersecurity
- Linux Operator
- Introduction to Python
- Linux System Administration

Possible professional roles after completing this module:

- NOC Operator
- IT Monitoring
- Infrastructure Support

Current status: **In Progress**

---

## Module 2 — SOC Entry (Blue Team Fundamentals)

Objective: learn security monitoring and incident detection.

Planned contents:

- Network Security

- SOC Operator

Skills developed:

- log analysis
- server hardening

- security monitoring

Potential roles:

- SOC Analyst L1

Status: **Planned**
